# 📊 Sales Analysis Dashboard

A web-based Sales Analysis Dashboard developed using **Python Flask**, **MySQL**, and **Pandas** for managing sales records and generating analytical reports.

## 🚀 Features

* 🔐 User Authentication
* 📦 Product Management
* 👥 Customer Management
* 💰 Sales Management
* 📈 Dashboard Analytics
* 📥 CSV Import
* 📤 CSV Export
* 🗄️ MySQL Database Integration

## 🛠️ Technologies Used

* Python
* Flask
* MySQL
* Pandas
* HTML
* CSS
* Git
* GitHub

## 🗄️ Database

**Database Name:** `sales_db`

## ⚙️ Installation

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Create Database

Run the SQL script:

```sql
database.sql
```

### Start Application

```bash
python app.py
```

## 🎯 Project Objectives

* Automate sales data management
* Improve reporting efficiency
* Generate analytical insights
* Simplify customer and product management
* Support CSV data import/export
